# Инструкция по добавлению продуктов

## Структура папок

```
your-website/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
├── images/              ← Сюда добавляйте изображения продуктов
│   ├── product1.jpg
│   ├── product2.jpg
│   └── ...
└── products/            ← Здесь данные о продуктах
    └── products.json
```

## Как добавить новый продукт

### Шаг 1: Добавьте изображение
1. Подготовьте фото продукта (рекомендуемый размер: 800x600px или соотношение 4:3)
2. Назовите файл, например: `product6.jpg`
3. Поместите в папку `images/`

### Шаг 2: Добавьте информацию о продукте
1. Откройте файл `products/products.json`
2. Добавьте новый блок в массив "products":

```json
{
  "id": 6,
  "name": "Название продукта",
  "description": "Краткое описание продукта",
  "image": "images/product6.jpg",
  "featured": false,
  "link": "#"
}
```

### Поля продукта:
- **id**: уникальный номер продукта (используйте следующий номер по порядку)
- **name**: название продукта (будет показано на карточке)
- **description**: описание продукта
- **image**: путь к изображению (всегда начинается с `images/`)
- **featured**: `true` = показывать в блоке "Новинки", `false` = только в каталоге
- **link**: ссылка на страницу продукта (пока можно оставить "#")

### Пример полного файла products.json:

```json
{
  "products": [
    {
      "id": 1,
      "name": "Глазурь Святая",
      "description": "Высококачественная глазурь",
      "image": "images/product1.jpg",
      "featured": true,
      "link": "#"
    },
    {
      "id": 2,
      "name": "Шоколадная глазурь",
      "description": "Натуральный шоколад",
      "image": "images/product2.jpg",
      "featured": true,
      "link": "#"
    }
  ]
}
```

## Важно!
- Всегда проверяйте, что JSON файл корректный (используйте jsonlint.com)
- Не забывайте запятые между продуктами
- Последний продукт в списке НЕ должен иметь запятую в конце
- Блок "Новинки" показывает первые 3 продукта с `featured: true`
- В каталоге показываются ВСЕ продукты

## Советы по изображениям:
- Формат: JPG или PNG
- Размер файла: не более 500KB для быстрой загрузки
- Соотношение сторон: 4:3 (например, 800x600px)
- Используйте понятные названия файлов: product1.jpg, product2.jpg и т.д.
